# Spring Boot開發個人網誌 Blog


## 採用技術框架 :
- 後端 backend：Spring Boot + JPA + thymeleaf 模板
- 資料庫 database：MySQL
- 前端UI frontend UI：Semantic UI framework

## 工具、環境：
- IntelliJ IDEA
- Maven 3
- JDK 8
- Axure RP 8
